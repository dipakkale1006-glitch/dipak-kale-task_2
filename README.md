# Laundry Wallah / Laundry Mart

This project is a simple website created using HTML. It is designed for a laundry service business where users can see available services, check prices, and enter their details to book a service.

# About the Website
The website gives basic information about Laundry Wallah / Laundry Mart. It explains what services are offered and helps customers understand pricing. It also includes a small form where users can enter their details.

# Features
- Displays a welcome message
- Shows an image related to laundry services
- Lists all available services like washing, ironing, and dry cleaning
- Provides a price list in table format
- Includes a simple booking form (Name, Email, Phone)
- Shows contact email information

# Technologies Used
- HTML (for structure of the website)

# File Included
- `index.html` → Main file of the website

# How to Use
1. Download or copy the project files
2. Open the `index.html` file in any web browser
3. View the website and its sections
4. Enter details in the form (not connected to backend)

# Note
- This is a basic project for learning purposes
- The form does not store data
- No CSS or JavaScript is used yet
